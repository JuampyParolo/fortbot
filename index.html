Defense in Depth

7 layers between the world
and your machine

Every message, every action, every data flow passes through deterministic validation before anything executes.

Layer 01
Trust Classification
Every message tagged: OWNER ‚Üí KNOWN ‚Üí UNKNOWN ‚Üí UNTRUSTED. Permissions cascade accordingly.
Layer 02
Privileged / Quarantined LLM
The planner never sees untrusted data. The quarantine LLM processes external input with zero tool access.
Layer 03
Taint Tracking
Data carries origin labels throughout the pipeline. Tainted values are blocked from reaching sensitive actions.
Layer 04
Policy Engine
Deterministic rules catch dangerous patterns before execution. No LLM judgment involved ‚Äî pure logic.
Layer 05
Guardian
Separate Python process provides semantic second opinion on sensitive actions. Fail-closed by design.
Layer 06
Executor Sandbox
40-command shell allowlist, file path restrictions, Docker isolation, plan timeout with automatic rollback.
Layer 07
Encryption at Rest
AES-256-GCM for the message database and credential vault. No plaintext secrets, ever.
